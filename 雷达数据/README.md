## 数据集4介绍
- 数据集名称：面向人体动作的毫米波雷达三维点云数据集
- 数据简要说明：本数据集是人体动作的毫米波雷达三维点云数据集（数据集简称为mmWave-3DPCHM-1.0），包括两种毫米波雷达传感器采集处理得到的点云数据。两种数据采集设备为TI公司的IWR1443-ISK和Vayyar公司的 vBlu 射频成像模组。本数据集包括毫米波雷达原始点云数据和预处理后的点云数据，并包含点云处理代码、各种预定动作的图片和视频等。
本数据集采集场景为一间会议室。数据采集过程使用到电脑、支架与毫米波雷达等。雷达安装在墙面上，离地面高度为1.75米。志愿者在一个矩形区域内完成规定动作，矩形区域距离雷达的水平距离为1.2米，长和宽均为2米。本数据集选取7名志愿者，采集12种人体动作，其中包含3个静态动作（跌倒、静坐和站立），9个动态动作（打拳、跳、左挥手、左前倾、开双臂、右挥手、右前倾、下蹲和步行）。每个动作采集3组数据，每组采集3分钟。在采集过程中，志愿者连续完成规定动作。数据保存为excel文件，具体格式如图4所示。为了便于研究者使用，我们对TI毫米波雷达的数据进行多帧融合聚类等预处理，处理的代码和结果都在对应文件夹内。

## 数据集8介绍
- 数据集名称：K波段超宽带雷达跌倒检测图谱数据集1.0
- 数据简要说明：K波段超宽带雷达跌倒检测图谱数据集1.0（K-UWB-MSHA-1.0），发布两个场景下36名受试者共3240例UWB雷达行为测量数据。该数据集通过K波段UWB雷达探测系统获得，雷达工作频段为22.8~24.8 GHz，信号体制为调频连续波信号。
该数据集包含了36名志愿者日常行为动作和跌倒动作的雷达数据，动作类型有行走、坐下、躺下、起立、慢速上下楼梯、拄拐行走、慢跑、爬行等20种常见日常行为动作和晕倒、跌倒、滑倒、上下楼梯绊倒等10种跌倒动作。数据采集的两个房间尺寸和场景布置均不一样，具体描述见论文。数据对应的实验方案通过了陆军军医大学伦理委员会批准（批准号：AF/sc-03/1.0，2021年）。场景一中30人的数据作为训练集，场景二中6人的数据作为测试集。雷达图谱数据段的持续时间均为4秒，对应的数据预处理方法见论文。

## 数据集10介绍
- 数据集名称：高分辨三维毫米波雷达数据集1.0
- 数据简要说明：高分辨三维毫米波雷达数据集(3DRIED)1.0，发布81组毫米波雷达三维成像回波数据及影像。该数据集获取模式为毫米波传感器沿二维平面扫描进行近场成像，扫描尺寸大小为0.4m×0.4m，雷达中心频率为78.8GHz，发射信号带宽为3.6GHz。毫米波传感器在距离向、水平向和垂直向的信号采样点数分别为256点、407点和200点，形成的单通道原始回波数据维度为256×407×200。

近场观测目标包括了刀具、枪械等金属目标，涵盖了单个目标、多个目标以及隐匿目标（纸盒、布袋）等环境条件。

本数据集中雷达传感器具有4个接收通道，最终数据集中原始回波数据维度为256×407×200×4。回波数据格式为mat（Matlab软件矩阵文件），同时提供了观测目标场景光学图像、二维与三维成像的参考影像。
